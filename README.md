# 英语单词打字学习游戏

这是一个基于 Next.js 的英语单词学习打字游戏。

## 技术栈

- Next.js 13+ (App Router)
- TypeScript
- Tailwind CSS
- React

## 开发

首先安装依赖:

```bash
pnpm install
```

然后运行开发服务器:

```bash
pnpm dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看应用.

## 构建

```bash
pnpm build
```

## 特性

- 英文单词显示
- 发音播放
- 中文释义展示
- 键盘快捷键支持

## Getting Started
